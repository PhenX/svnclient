SVN Client
=========
SVN client written in PHP, uses SVN executables.
This project is a WIP, a lot more is coming.

Usage
=====
`>> php console.php svn:update /path/to/working/copy`

Available commands
------------------
 * `svn:init`
 * `svn:update`
 * ... see `help` command
